# Examples

This repository is the **`fastmvc-cli`** package (the `fast` command). **Sample applications** are not vendored here—they are produced **on your machine** when you run:

```bash
fast quickstart --name my_api
# or
fast generate
```

That output is the best “official” example: it matches the **fastx-mvc** template the CLI copies from your environment.

## Quick path

1. Follow **[docs/GETTING_STARTED.md](../docs/GETTING_STARTED.md)**.
2. Open the generated project README and `app` entrypoint.
3. Use **`fast add resource`** inside that tree to scaffold new API operations.

For curated showcase repos (if/when published), see the **Related repositories** section in the main [README](../README.md).
